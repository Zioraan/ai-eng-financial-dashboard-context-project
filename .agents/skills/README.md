# Skills Directory

This directory stores custom Copilot agent skills for this repository.

## Structure

Each skill should be a folder with a `SKILL.md` file:

- `.agents/skills/<skill-name>/SKILL.md`

Optional supporting files can live beside `SKILL.md` (examples, templates, checklists, etc.).

## Authoring Notes

- Keep each skill focused on one workflow.
- Include clear usage criteria (when to use and when not to use).
- Prefer concise, actionable steps.
- Keep repository-specific conventions aligned with `.agents/rules` and `memory-bank`.

## Starter

A starter skill template is available at:

- `.agents/skills/example-project-skill/SKILL.md`
